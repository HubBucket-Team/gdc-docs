# Biospecimen Data #
## Description ##
Biospecimen data describe biospecimen-related elements. A biospecimen element is a classification of biospecimen data that describe a common form of biospecimen. A biospecimen often takes the form of a case, sample, portion, analyte or aliquot. They can be seen in a hierarchical relationship to one another.
